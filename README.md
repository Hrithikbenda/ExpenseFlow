# ExpenseFlow 💰

ExpenseFlow is a smart expense tracking application built using React, Spring Boot, and MySQL.

## Features

- User Authentication (Login/Register)
- Add, Edit & Delete Expenses
- Expense Notes
- Expense Analytics
- Monthly Budget Tracker
- AI Financial Coach
- PDF Report Export
- User Settings
- Responsive UI

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Axios
- Recharts
- jsPDF

Backend:
- Spring Boot
- Spring Data JPA
- MySQL
